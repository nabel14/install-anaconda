# Install Anaconda

Dibuat untuk memenuhi tugas Machine Learning.

## Langkah-langkah Instalasi Anaconda
1. Membuka website resmi Anaconda.
2. Mendownload installer Anaconda sesuai sistem operasi.
3. Menjalankan file installer yang sudah didownload.
4. Mengikuti proses instalasi sampai selesai.
5. Membuka Anaconda Navigator untuk memastikan instalasi berhasil.
